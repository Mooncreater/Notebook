## 🧭 学习路线

```
凯撒密码 → Base64 → AES 对称加密 → RSA 非对称加密 → HTTPS 综合实战
  (入门)    (编码)      (真加密)        (双钥)           (全部串联)
```

**建议按顺序阅读**，每一章都是下一章的基础。

---

## 🛠️ 需要的工具

- **PowerShell**（Windows 自带，用于 Base64 编解码）
- **浏览器 F12**（看 HTTPS 证书、Network 请求）
- **OpenSSL**（可选，用于 AES 实战。装 Git 时会自带，在 Git Bash 里用）

---

## 💡 核心脉络

整个笔记围绕一条主线展开：

> **你怎么知道在和真正的百度聊天，而不是一个长得像百度的骗子？**

答案藏在 5 个章节中：

1. **凯撒密码**：加密是什么感觉 → 但太弱了
2. **Base64**：有些东西不是加密，只是换马甲
3. **AES**：真正的加密，全世界都在用 → 但密钥怎么给对方？
4. **RSA**：双钥解决密钥分发 → 但 RSA 怎么防冒充？
5. **HTTPS**：证书 + RSA + AES 全用上，完美解决

---

## 📝 使用方式

每章约 10-15 分钟读完。读完一章后：
1. 做章末的练习题
2. 运行里面的 PowerShell / OpenSSL 命令
3. 把不懂的地方标出来，回头再问
