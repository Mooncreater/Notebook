# GPU

为什么要GPU？

- 需要更大的算力

## GPU与CPU的区别

CPU :

- 较少的复杂核  
- 大量缓存  
- 大且慢的内存

![alt text](image.png)

GPU :

- 大量简单核    
- 小缓存低延时  
- 小且快的内存

![alt text](../img/{6DB5B881-EBAF-4E90-A684-CCBE14E69369}.png)

## GPU计算
步骤：

1. CPU 向 GPU 传输数据
2. GPU 核内进行计算
3. GPU 将结果传回 CPU
