# 光栅图形学与二维图元算法	
	
将几何图元（直线、圆、多边形）转换为屏幕像素的过程称为**光栅化**（Rasterization）。	
	
!!! tip "核心要点"	
    光栅化的核心是**用整数运算替代浮点运算**。Bresenham 算法和 Midpoint 算法是这一思想的典范。	
	
## 1. 画直线	
	
### DDA 算法（Digital Differential Analyzer）	
	
用增量方式沿直线步进：	
	
$$y_{k+1} = y_k + m,\quad m = \frac{\Delta y}{\Delta x}$$	
	
- 每次 $x$ 加 1，$y$ 加斜率 $m$	
- 需要浮点运算和取整	
- $|m| > 1$ 时交换 $x$、$y$ 角色	
	
!!! warning "问题"	
    DDA 依赖浮点加法和取整，每步都需要，效率较低。	
	
### Bresenham 算法	
	
**仅用整数加减和移位**，无需浮点运算，效率极高。	
	
核心思想：维护一个**决策参数** $p_k$，判断下一个像素是走 $x$ 方向还是斜方向。	
	
对 $0 < m < 1$：	
	
初始：$p_0 = 2\Delta y - \Delta x$	
	
迭代：	
$$p_{k+1} = \begin{cases} p_k + 2\Delta y - 2\Delta x & \text{if } p_k \geq 0 \\ p_k + 2\Delta y & \text{if } p_k < 0 \end{cases}$$	
	
- $p_k \geq 0$：右上像素 $(x_k+1, y_k+1)$	
- $p_k < 0$：右像素 $(x_k+1, y_k)$	
	
!!! danger "考试重点"	
    Bresenham 的决策参数递推公式是高频考点，理解 $p_k$ 的几何含义：$p_k = 2\Delta y \cdot x_k - 2\Delta x \cdot y_k + const$。	
	
## 2. 画圆	
	
### 中点画圆算法（Midpoint Circle）	
	
利用圆的**八对称性**，只需计算 $1/8$ 圆弧。	
	
对圆心在原点、半径为 $R$ 的圆：	
	
圆方程：$f(x, y) = x^2 + y^2 - R^2 = 0$	
	
从 $(0, R)$ 开始，决策参数：	
$$p_0 = \frac{5}{4} - R \;\approx\; 1 - R \quad\text{（取整）}$$	
	
迭代：若 $p_k < 0$，选 $(x_k+1, y_k)$；否则选 $(x_k+1, y_k-1)$。	
	
利用八对称点 $( \pm x, \pm y)$ 和 $( \pm y, \pm x)$ 一次性画出完整圆。	
	
## 3. 多边形填充	
	
### 扫描线填充（Scan-Line Fill）	
	
逐行扫描，计算每条扫描线与多边形的交点，两两配对填充。	
	
步骤：	
1. 建立边表（Edge Table），记录每条边与扫描线的交点	
2. 从下往上逐行处理，维护活动边表（Active Edge Table）	
3. 交点按 $x$ 排序，奇偶配对填充	
	
### 奇偶规则（Even-Odd Rule）	
	
从点沿扫描线方向计数与边的交点数：	
- 奇数 :arrow_right: 在多边形内部	
- 偶数 :arrow_right: 在多边形外部	
	
## 4. 反走样（Anti-Aliasing）	
	
走样（Aliasing）是离散采样导致的锯齿、摩尔纹等伪影。	
	
### 常用方法	
	
- **超采样（Super-Sampling）**：以更高分辨率渲染再降采样	
- **区域采样（Area Sampling）**：像素值 = 图元覆盖该像素的面积比例	
- **MSAA（Multi-Sample Anti-Aliasing）**：每个像素多次采样深度/模板，着色只算一次，GPU 硬件支持	
	
## 5. 参数式方程图形	
	
用参数 $t$ 表示曲线上的点：	
	
$$\begin{cases} x = f(t) \\ y = g(t) \end{cases}, \quad t \in [0, 1]$$	
	
- **Bézier 曲线**：由控制点定义的参数曲线	
- 二次 Bézier：$B(t) = (1-t)^2 P_0 + 2(1-t)t P_1 + t^2 P_2$	
- 三次 Bézier：$B(t) = (1-t)^3 P_0 + 3(1-t)^2 t P_1 + 3(1-t)t^2 P_2 + t^3 P_3$	
	
## 6. 隐式方程	
	
形式 $f(x, y) = 0$ 表示的图形：	
- 直线：$Ax + By + C = 0$	
- 圆：$(x - x_c)^2 + (y - y_c)^2 - R^2 = 0$	
- 椭圆：$\frac{(x - x_c)^2}{a^2} + \frac{(y - y_c)^2}{b^2} - 1 = 0$	
	
!!! tip "优势"	
    隐式方程判断点与图形的关系：$f(x, y)$ 的符号直接表示点在图形内部/边界/外部，非常高效。
