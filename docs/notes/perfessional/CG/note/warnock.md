## 消影

### Warnock算法
Warnock算法是由John Warnock在1969年提出的一种用于隐藏面消除的算法。该算法基于分治策略，将场景划分为更小的区域，并递归地处理这些区域以确定可见性。


### Z-Buffer算法

帧缓存  深度缓冲区

Z-Buffer算法是一种基于深度缓冲区的隐藏面消除算法。它通过为每个像素存储一个深度值来确定哪个表面在视线方向上更近，从而实现正确的可见性处理。
- 初始化帧缓存和深度缓存区
- 对于每一个多边形得到的像素和深度值对比，更新帧缓存和深度缓存区
- 最终帧缓存中的颜色值即为最终图像

### painter's algorithm 画家算法

思路：从远到近进行画图，后画的覆盖前面画的

- 进行一个深度排序，按照从远到近的顺序对多边形进行排序
- 按照排序的顺序依次绘制多边形


问题：
- [x] 可能存在循环重叠的状态 

-> 解决方法：将多边形进行切割，直到不存在循环重叠的状态

**BSP树**：二叉空间分割树（Binary Space Partitioning Tree） 
选择一个多边形作为切割平面，将空间划分为两个子空间，递归地对每个子空间进行切割，直到所有多边形都被划分到叶节点中。这样可以确保在绘制时按照正确的顺序进行处理，避免循环重叠的问题。 

根据视角对于根节点的前后关系进行递归遍历BSP树绘制

🌲的构造可以预先构造，不需要实时渲染
