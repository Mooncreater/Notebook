## 数据集成

数据集成是将多个数据源的数据进行整合，形成一个统一的数据视图的过程。        
数据集成的目的是为了消除数据冗余、提高数据质量和一致性，从而为后续的数据分析和挖掘提供可靠的数据基础。
 

## 数据降维 Data Reduction

![alt text](../img/image.png)

随着维度增高，数据的稀疏性增加，距离度量失效，模型过拟合等问题都会出现，这就是所谓的维度灾难（Curse of Dimensionality）。   

数据降维的常用方法包括：  

-   主成分分析（PCA）：通过线性变换将数据投影到一个新的坐标系中，使得投影后的数据具有最大的方差。**中心化** -> **协方差矩阵** -> **特征值分解** -> **选择前k个特征向量** -> **数据投影**    
-   Grid -Indexing：将数据空间划分为网格，每个网格包含一定数量的数据点，从而减少数据的维度。    
-   归一化 ：Min-Max Scaling: 将数据缩放到一个指定的范围内，通常是[0, 1]，以减少数据的维度。
