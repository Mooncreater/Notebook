# 分类器	
	
分类是有监督学习方法，通过学习带标签的训练数据来预测新样本的类别。	
	
## 1. KNN 分类器	
	
**K 近邻（K-Nearest Neighbors）**：找到最近的 K 个训练样本，多数投票决定类别。	
	
- 距离度量：欧氏距离 $d = \sqrt{\sum (x_i - y_i)^2}$、曼哈顿距离、余弦相似度	
- K 值选择：太小易过拟合，太大易欠拟合，常用交叉验证确定	
- 优点：无需训练，可处理多分类	
- 缺点：预测慢（需计算所有距离），对维度灾难敏感	
	
## 2. 决策树分类器	
	
根据信息增益选择最优特征递归划分，构建树结构。	
	
### 信息熵	
	
$$H(D) = -\sum_{i=1}^{n} p_i \log_2 p_i$$	
	
### 信息增益	
	
$$IG(D, A) = H(D) - H(D|A) = H(D) - \sum_{v} \frac{|D_v|}{|D|} H(D_v)$$	
	
选择 IG 最大的特征进行划分。	
	
### 增益率（C4.5）	
	
$$GainRatio(D, A) = \frac{IG(D, A)}{SplitInfo(D, A)}$$	
	
### Gini 指数（CART）	
	
$$Gini(D) = 1 - \sum_{i=1}^{n} p_i^2$$	
	
- 优点：可解释性强，能处理数值和类别特征	
- 缺点：容易过拟合，需剪枝（预剪枝/后剪枝）	
	
## 3. 朴素贝叶斯分类器	
	
基于贝叶斯定理 + 特征条件独立假设：	
	
$$P(C_k | X) = \frac{P(X | C_k) \cdot P(C_k)}{P(X)}$$	
	
由于 $P(X)$ 对所有类别相同：	
	
$$\hat{y} = \arg\max_{k} P(C_k) \prod_{j=1}^{m} P(x_j | C_k)$$	
	
- **拉普拉斯平滑**：避免概率为零，$P(x_j | C_k) = \frac{count(x_j, C_k) + 1}{count(C_k) + |V|}$	
- 优点：计算快，适合高维数据	
- 缺点：特征独立假设通常不成立	
	
## 4. 支持向量机（SVM）	
	
寻找最大间隔超平面来分离不同类别。	
	
### 线性可分 SVM	
	
优化目标：	
	
$$\min_{w, b} \frac{1}{2}\|w\|^2 \quad \text{s.t.} \quad y_i(w^T x_i + b) \geq 1$$	
	
**支持向量**：落在间隔边界上的样本。	
	
### 软间隔 SVM	
	
引入松弛变量 $\xi_i \geq 0$ 允许少量误分类：	
	
$$\min_{w, b, \xi} \frac{1}{2}\|w\|^2 + C\sum_{i=1}^{n}\xi_i$$	
	
$C$ 越大 → 对误分类惩罚越大 → 间隔越小。	
	
### 核函数（Kernel Trick）	
	
将数据映射到高维空间，使非线性可分数据变得线性可分：	
	
- **线性核**：$K(x_i, x_j) = x_i^T x_j$	
- **多项式核**：$K(x_i, x_j) = (x_i^T x_j + c)^d$	
- **RBF 核**：$K(x_i, x_j) = e^{-\gamma\|x_i - x_j\|^2}$	
	
## 5. 模型评估	
	
### 混淆矩阵	
	
|  | 预测正类 | 预测负类 |	
|--|----------|----------|	
| 实际正类 | TP | FN |	
| 实际负类 | FP | TN |	
	
- **准确率（Accuracy）**：$\frac{TP + TN}{TP + TN + FP + FN}$	
- **精确率（Precision）**：$\frac{TP}{TP + FP}$	
- **召回率（Recall）**：$\frac{TP}{TP + FN}$	
- **F1 Score**：$\frac{2 \cdot P \cdot R}{P + R}$	
	
### ROC 与 AUC	
	
- ROC 曲线：以 $TPR = \frac{TP}{TP+FN}$ 为纵轴，$FPR = \frac{FP}{FP+TN}$ 为横轴	
- AUC：ROC 曲线下面积，越接近 1 越好	
	
### 交叉验证	
	
K 折交叉验证：将数据分成 K 份，K-1 份训练，1 份验证，轮换 K 次取平均。	
	
## 6. 过拟合与欠拟合	
	
- **过拟合**：训练误差低，测试误差高 → 剪枝、正则化、增加数据	
- **欠拟合**：训练误差和测试误差都高 → 增加特征、增加模型复杂度
