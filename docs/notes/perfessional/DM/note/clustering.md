# 聚类

## 概念
聚类是一种无监督学习方法，旨在将数据集划分为若干个簇，使得同一簇内的数据点相似度较高，而不同簇之间的数据点相似度较低。

## K-means算法

1. 初始化：随机选择K个数据点作为初始簇中心。    
2. 分配：将每个数据点分配到距离最近的簇中心。   
3. 更新：重新计算每个簇的中心，返回第二步。 
4. 停止：直到簇中心不再变化或达到最大迭代次数。 

## density-based clustering

基于密度的聚类方法，如DBSCAN，通过寻找密度较高的区域来形成簇，能够识别任意形状的簇，并且对噪声具有鲁棒性。

## DBSCAN 算法
1. 选择一个未访问的数据点，标记为核心点。
2. 从核心点出发，找到所有密度可达的点，形成一个簇。
3. 重复上述步骤，直到所有数据点都被访问过。


## 层次聚类

1.  每个点作为一个簇
2.  合并最近两个簇
3.  重复直到所有点在一个簇中

## 高级聚类场景

### soft clustering
可以属于多个分类  

- 苹果可以属于绿色和红色两类

相比于硬聚类，多了一个权重

### 基于 Mixture Model

概率分布拟合，类比k-means k个高斯分布，EM算法求解

#### EM算法

1. E-step：计算每个数据点属于每个簇的概率。     
2. M-step：根据计算的概率重新估计簇的参数（如均值和协方差）.

### Spectral Clustering
基于图论的聚类方法，通过构建数据点之间的相似度矩阵，并计算其特征值和特征向量来进行聚类，适用于非线性可分的数据。
