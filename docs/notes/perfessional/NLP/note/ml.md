# 深度学习基础	
	
机器学习 ≈ 寻找一个函数。深度学习 = 用深层神经网络定义函数族，并用反向传播 + 梯度下降找到最优函数。	
	
!!! tip "核心要点"	
    深度学习三步走：**定义函数族**（神经网络）→ **评价好坏**（损失函数）→ **选择最佳**（梯度下降）。反向传播利用链式法则高效计算梯度。	
	
## 1. 机器学习三步框架	
	
| 步骤 | 含义 | 深度学习对应 |	
|:---|:---|:---|	
| Step 1 | 定义一组函数 (A set of function) | 设计神经网络架构 |	
| Step 2 | 评价函数好坏 (Goodness of function) | 定义损失函数 |	
| Step 3 | 选择最佳函数 (Pick the best) | 梯度下降优化 |	
	
## 2. 神经网络	
	
### 神经元 (Neuron)	
	
$$z = a_1w_1 + a_2w_2 + \cdots + a_Kw_K + b$$	
$$a = \sigma(z)$$	
	
- **权重 (weights)** $w$：控制每个输入的重要程度	
- **偏置 (bias)** $b$：控制激活的阈值	
- **激活函数** $\sigma$：引入非线性	
	
### 常见激活函数	
	
| 函数 | 公式 | 适用场景 |	
|:---|:---|:---|	
| **Sigmoid** | $\sigma(z) = \frac{1}{1+e^{-z}}$ | 二分类输出层 |	
| **Softmax** | $\text{Softmax}(z_i) = \frac{e^{z_i}}{\sum_j e^{z_j}}$ | 多分类输出层 |	
| **ReLU** | $\max(0, x)$ | 隐藏层（解决梯度消失） |	
| **Tanh** | $\frac{e^{2x}-1}{e^{2x}+1}$ | RNN/LSTM 中常见 |	
	
!!! warning "常见误区"	
    Sigmoid 用于二分类输出层，Softmax 用于多分类输出层。隐藏层现在基本用 ReLU，因为能解决梯度消失问题。	
	
### 全连接前馈网络 (Fully Connected Feedforward Network)	
	
- 输入层 → 隐藏层（多层）→ 输出层	
- 每层神经元与下一层全连接	
- Deep = 很多隐藏层	
	
### 为什么需要深？	
	
相同参数量下，**深而窄**比**浅而宽**更高效：深层网络能用更少的"片段"表达复杂函数，参数利用率更高。	
	
网络深度演进：AlexNet (8层, 2012) → VGG (19层, 2014) → GoogleNet (22层, 2014) → ResNet (152层, 2015)	
	
## 3. 损失函数	
	
| 任务类型 | 损失函数 | 公式 |	
|:---|:---|:---|	
| **回归** | MSE | $L = \frac{1}{n}\sum(y_i - \hat{y}_i)^2$ |	
| **二分类** | 二元交叉熵 | $L = -\frac{1}{n}\sum[y_i\log\hat{y}_i + (1-y_i)\log(1-\hat{y}_i)]$ |	
| **多分类** | 分类交叉熵 | $L = -\frac{1}{n}\sum y_i \log \hat{y}_i$ |	
	
总损失是对所有训练样本损失求和：希望总损失尽可能小。	
	
## 4. 梯度下降 (Gradient Descent)	
	
### 基本公式	
	
$$w_{t+1} = w_t - \eta \cdot \frac{\partial L}{\partial w}$$	
	
- $\eta$：学习率。太大震荡不收敛，太小收敛太慢	
- 随机初始化参数 → 沿负梯度方向更新 → 收敛或停止	
	
### 随机 vs 批量	
	
- **BGD**：用全部数据算梯度，准确但慢	
- **SGD**：每次随机抽一个 batch 算梯度，更快更常用	
- **Mini-batch GD**：折中方案	
	
### 局部最小值问题	
	
- 梯度下降不保证全局最优，不同初始点可能收敛到不同局部最小值	
- 在实践中，高维空间的局部最小值往往质量接近	
- 鞍点 (saddle point) 比局部最小值更常见，是更大的挑战	
	
## 5. 反向传播 (Backpropagation)	
	
链式法则逐层计算梯度，从输出层向输入层传播：	
	
$$\frac{\partial L}{\partial w} = \frac{\partial L}{\partial z} \cdot \frac{\partial z}{\partial w}$$	
	
核心：**前向计算激活值，反向传播梯度**。是训练神经网络最高效的方法。	
	
## 6. 深度学习理论前沿	
	
### Neural Tangent Kernel (NTK)	
	
在无限宽极限下，深度神经网络退化为核方法（NTK / NNGP）。这解释了深度网络为何能泛化：本质上在做一个特定的核回归。	
	
### 过参数化理论	
	
过参数化模型反而更容易优化 (Du & Lee, 2018)。深度网络巨大的参数量不是负担，而是优势。	
	
!!! danger "考试重点"	
    三步框架、激活函数选择、梯度下降公式、反向传播的链式法则、局部最小值与鞍点的区别。
