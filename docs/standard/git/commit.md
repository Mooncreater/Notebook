## 类型: feat/fix/docs/style/refactor/perf/test/chore

| 类型 | 含义 | 示例 |
|:---|:---|:---|
| **feat** | 新功能 | `feat: 添加用户登录功能`<br>`feat(搜索): 支持全文搜索` |
| **fix** | 修复 Bug | `fix: 修复首页加载超时问题`<br>`fix(API): 修复用户接口返回404错误` |
| **docs** | 文档更新 | `docs: 更新项目README`<br>`docs: 添加部署说明文档` |
| **style** | 代码格式 | `style: 格式化HTML代码`<br>`style(css): 调整代码缩进` |
| **refactor** | 代码重构 | `refactor: 重构用户验证逻辑`<br>`refactor(组件): 将大组件拆分成小组件` |
| **perf** | 性能优化 | `perf: 优化图片加载速度`<br>`perf(前端): 实现路由懒加载` |
| **test** | 测试相关 | `test: 添加登录功能单元测试`<br>`test(e2e): 添加用户注册流程测试` |
| **chore** | 杂务/配置 | `chore: 更新依赖包版本`<br>`chore: 配置eslint规则` |
| **revert** | 回滚 | `revert: 回退登录功能修改`<br>`revert: "feat: 添加用户登录功能"` |
| **merge** | 合并分支 | `merge: 合并develop分支到main` |
| **build** | 构建相关 | `build: 修改构建脚本`<br>`build: 更新打包配置` |
| **ci** | 持续集成 | `ci: 配置GitHub Actions`<br>`ci: 修改Jenkins构建脚本` |

#### 示例:  git commit -m"feat: 添加用户登录功能"
